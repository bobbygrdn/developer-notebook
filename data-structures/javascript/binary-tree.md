## Rules of BT

- Tree has only 1 root node
- Tree has only 1 path to each node
- Tree has maximum 2 nodes attached to each node

|---| Time Complexity | Space Complexity |
| Insertion | O(N) | O(N^0.5) |
| Search | O(N) | O(N^0.5) |
| Deletion | O(N) | O(N^0.5) |
